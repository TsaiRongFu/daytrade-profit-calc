📈 **台股當沖 / 波段交易損益平衡價計算器**

協助你在下單前快速確認：
- 💰 實際損益

## ✨ 功能特色

- ✅ 台股交易適用（TWD）
- ✅ 當沖 / 波段皆可使用
- ✅ 支援手續費 % 計算
- ✅ 即時計算損益與回本價
- ✅ 手機 / 桌機皆適用
- ✅ 純前端，GitHub Pages 部署

## 🧮 適合誰使用？

- 📊 台股當沖交易者  
- 📈 波段操作、想精準抓回本點  
- 🧠 不想每次都手算或開 Excel  
- ⏱️ 下單前快速確認「賣多少才不賠」
